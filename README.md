## 🌐 i8i - Advanced URL Shortener
**i8i** is a powerful URL shortener that simplifies link sharing with custom URLs, detailed analytics, and security options.

## 🚀 Features
- 🔗 Custom short links (e.g., `i8i.com/custom`).
- 📊 Analytics: clicks, location, devices.
- 🔒 Password protection for links.
- 🕒 Expiration settings for temporary links.
- 📌 QR code generation.
- 📡 API for developers.

## 📦 Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/i8i.git
   ```
2. Install dependencies:
   ```bash
   cd i8i && npm install
   ```
3. Start the server:
   ```bash
   npm run start
   ```
